# README

Demonstrate some examples of string manipulation and variables

## Examples

### Bash

```sh
# Default values ${1-defaults} 
./bash_strings.sh

# Also does substring matching "hello*"
./bash_strings.sh hello world
```

### Zsh

```sh
# Default values ${1-defaults} 
./zsh_strings.sh

# Also does substring matching "hello*"
./zsh_strings.sh hello world
```
