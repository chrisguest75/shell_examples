# README

Example textual diffing tools.  

REF: Binary Diffing Examples [here](../68_difftools/README.md)  

## VSCode

```sh
code --diff file1.txt file2.txt
```

## Beyond Compare

```sh
bcompare file1.txt file2.txt
```

## diff

```sh
diff file1.txt file2.txt
```
