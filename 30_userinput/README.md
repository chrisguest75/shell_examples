# README
Demonstrate how to use user input in scripts

* Readline
* Integer input verification
* Read secret value
* Handle yesno question. 

## Examples
```sh
# Answer questions and use as input
./questions.sh
```

```sh
# Have an infinite loop and listen for keys
./quit_on_keypress.sh
```
