# OH MY SHELLS

The shell themes.

## Oh my zsh

ohmyzsh/ohmyzsh repo [here](https://github.com/ohmyzsh/ohmyzsh)  

```sh
# path to oh-my-zsh
echo $ZSH

# oh-my-zsh options
omz --help

# force it to update and sync
omz update
```

### Troubleshooting OMZ

The `.git` folder was deleted.  

```sh
git clone https://github.com/ohmyzsh/ohmyzsh.git .oh-my-zsh
```
